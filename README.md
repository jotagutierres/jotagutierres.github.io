# Carlos Gutierres — Product Designer

Personal portfolio · São Paulo, BR

**Live:** [jotagutierres.github.io](https://jotagutierres.github.io)

Built as a single-page static site with React (via Babel Standalone) + custom CSS.
No build step — every file in this repo is what ships to the browser.

## Stack

- HTML · CSS · React (inline JSX, transpiled in-browser via `@babel/standalone`)
- Google Fonts — Inter Tight, Instrument Serif, JetBrains Mono
- Zero dependencies, zero build pipeline

## Structure

```
index.html          → main portfolio page
styles.css          → global design tokens + sections
case-study.css      → shared styles for /work/* pages
src/                → JSX components (loaded via <script type="text/babel">)
work/               → individual case study pages
images/             → portrait + assets
```

## Case studies

- **FordPass CRM** — connected vehicle activation in Brazil
- **FordPass®** — connected services app for Ford owners
- **Coral Brazil** — 70-year-old paint brand reimagined
- **Tele Sena** — rebuilding Brazil's iconic bond platform
- **Ranger Scroll Drive** — pandemic-era virtual test drive
